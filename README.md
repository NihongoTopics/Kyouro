このリポジトリは、**教育ローマ字（Kyouro）** プロジェクトの2つの主要な著作を公開しています。

- **Proposing a New Romanization System of Japanese**  
  → 新しいローマ字体系（Kyouro）の提案

- **Semantics of Tokyo Japanese**（本PDF）  
  → 日本語の意味論。Kyouroの言語理論の一部として、ga・mo・wa・shika・dakeoなどの助詞を統一的に説明する枠組み

Semantics of Tokyo Japanese はローマ字の**綴り規則自体**には依存していません（独立した意味論）。  
一方、教育ローマ字（Kyouro）は単なる綴りの提案ではなく、**意味論を含む言語理論全体**として設計されています。  
2冊は相互に補完する関係にあります。
